# 🌾 Conexão Campo-Cidade: Calculadora de Sementes

## 📋 Sobre o Projeto
[cite_start]Este projeto foi desenvolvido como participante do **Concurso Agrinho 2026** na categoria **Desenvolvimento Front-End**[cite: 1, 5]. [cite_start]O site é uma aplicação autoral que busca explorar o tema **"Festejando a Conexão Campo-Cidade"**, demonstrando como a produção do campo está diretamente ligada ao cotidiano e ao abastecimento das cidades[cite: 5, 7, 8].

[cite_start]A proposta do site é conscientizar a pessoa usuária sobre a importância da agricultura e oferecer uma ferramenta interativa: uma **Calculadora de Sementes**[cite: 95, 97, 198]. Com ela, moradores urbanos que desejam criar hortas comunitárias ou domésticas, assim como pequenos produtores rurais, podem planejar o plantio calculando a quantidade necessária de sementes com base na área disponível em metros quadrados.

---

## 🚀 Funcionalidades
* [cite_start]**Interface Dinâmica e Interativa:** Uso de JavaScript para realizar cálculos em tempo real sem a necessidade de recarregar a página[cite: 75, 77, 89].
* [cite_start]**Validação de Dados:** O sistema valida as entradas do usuário, impedindo cálculos com valores vazios, negativos ou iguais a zero[cite: 82].
* [cite_start]**Design Responsivo:** A interface foi projetada utilizando Media Queries, garantindo que o site funcione perfeitamente em computadores, tablets ou celulares[cite: 169, 173, 177, 178, 179].
* [cite_start]**Acessibilidade Digital:** Uso de HTML Semântico (como `header`, `main`, `section` e `footer`) para facilitar a leitura por leitores de tela [cite: 150, 152, 153, 154, 155, 159][cite_start], além do atributo `aria-live` para anunciar dinamicamente o resultado do cálculo[cite: 181].
* [cite_start]**Identidade Visual Temática:** Cores selecionadas intencionalmente, utilizando tons de verde para representar a agricultura/campo e tons de azul para simbolizar a tecnologia/conexão[cite: 125, 128, 130, 131].

## 🛠️ Tecnologias Utilizadas
[cite_start]O projeto foi construído do zero utilizando as três tecnologias fundamentais do Front-End[cite: 34, 35, 112]:
1. [cite_start]**HTML5:** Para a estruturação semântica de todo o conteúdo da página[cite: 36, 37, 150].
2. [cite_start]**CSS3:** Para a estilização, definição da identidade visual e regras de responsividade[cite: 57, 58, 125, 162].
3. [cite_start]**JavaScript:** Para a construção da lógica de cálculo e interatividade com a pessoa usuária[cite: 73, 74, 75].
